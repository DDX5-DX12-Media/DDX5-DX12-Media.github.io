This Page Will Explain The Book Scripting Used Inside Of Derm Version 2.0.3 Or Higher


The Script Uses Each Line Like A Seperate Value. On The Top Page You Will Need To Enter An Accepted Command

The Commands For The Lock Block Are SetCode And Passcode, SetCode Is Used To Set The Code That The Block Is Locked With, Passcode Is Used To Verify If The Codes Match

The Second Line For Both Commands Need To Be The Code. Using Passcode Command It Will Check If The Code In The Book Matched The Bocks Code, If They Match Redstone Pulse Is Emitted

Only The First Page Is Used In The Scripting At This Moment
