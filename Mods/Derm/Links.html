This Page Will Explain The Link Block Version 2.0.4 Or Higher


Theese Blocks Are Redstone Enabled Blocks, That Are Linked By The Integrated Scripting Explained In The Books Link

There Are 2 Types Of Theese Blocks, Sender And Receiver

The Sender Is The Block That The Script Is Attatched Onto, To Use This Block You Need To Link It First Then Power It With Redstone Or Use The Activate Command

The Reciever Is The Block That Toggles Its Redstone Power State Like A Lever And Can Be Activated Anywhere In The Same World, (Overworld, Nether ect)
